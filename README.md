## Installation

```
npm install @tethys/store --save
# or if you are using yarn
yarn add @tethys/store
```

## Development
```base
$ git clone https://github.com/tethys-org/store

$ cd store && yarn

$ yarn start:docs // open http://localhost:8887
```

## Release & Publish

```
yarn release
yarn pub
```
